Hexo搭建博客教程

发表于 2017-05-14 |

准备工作

  • 下载node.js并安装（官网下载安装），默认会安装npm。
  • 下载安装git（官网下载安装）
  • 下载安装hexo。方法：打开cmd 运行npm install -g hexo（要翻墙）
  • 创建github仓库，用于存储我们的网站资源。
  • 在GitHub上创建名为xxx.github.io的项目，xxx是自己GitHub的用户名

本地搭建hexo静态博客

  • 命令行cd到需要存放的目录。
  • 生成hexo模板：
    $ hexo init xxx.github.io。xxx可以使用自己GitHub的用户名。运行完成后，会创建一个名为xxx.github.io的文件夹，里面包含hexo模板资源。
  • cd到xxx.github.io目录，运行hexo server，运行成功后访问本地http://localhost:4000/可以看到博客已经搭建完成。

  • 运行：npm install hexo-deployer-git --save

  • 运行：hexo g（本地生成静态文件）
  • 运行：hexo d（将本地静态文件推送至GitHub）

此时，打开浏览器，可以访问http://yy-linye.github.io

绑定域名

现在我们已经可以通过GitHub的域名访问我们的博客，不过很多时候我们需要使用自己的域名，这时候我们就需要进行域名绑定。

  • 我们先去购买一个域名，这里我们使用的是阿里云：http://wanwang.aliyun.com

  • 获取GitHubPage的 ip 地址：ping http://yy-linye.github.io

  • 域名绑定 ip 地址

  • 在阿里云登录你的账号，找到域名相关选项
  • 点击解析进行域名解析：
  • 输入需要绑定的 ip 地址

  • 绑定成功后，如下图所示

  • 博客添加CNAME文件

  • 配置完域名解析后，进入博客目录，在source目录下新建CNAME文件（CNAME文件需要去除后缀），写入域名，如：izaya.cn
  • 运行：hexo g
  • 运行：hexo d

至此，绑定域名就完成了，我们现在可以通过 izaya.cn 来访问博客。

安装主题

以next主题为例。进入themes目录，进行如下操作：

  • git clone https://github.com/iissnan/hexo-theme-next.git（主题的地址）
  • 将下载的主题文件夹名改为next（主题的名字）
  • 打开根目录下的_config.yml文件，修改themes参数为next（主题文件夹的名字）
  • 运行：hexo g
  • 运行：hexo d

现在打开 izaya.cn（你自己的域名），会发现你的博客已经焕然一新。

关于hexo-next主题的更多个性化配置，可以参考：next主题使用

编写博客

  • 在xxx.github.io目录下执行hexo new "我的文章"，会在source->_posts文件夹内生成一个.md文件。
  • 编辑该文件（遵循Markdown规则）
  • 运行：hexo clean （删除本地静态文件，可不执行）
  • 运行：hexo g（生成本地静态文件）
  • 运行：hexo d（将本地静态文件推送至github）

错误解决

permission denied

如果在执行hexo d时报错，在前面加上sudo进行提权，sudo hexo d这样就可以了。
