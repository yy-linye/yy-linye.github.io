快捷键

视图

  • 切换实时预览：Shift + Cmd + I
  • 切换单词计数器：Shift + Cmd + W
  • 切换透明度：Shift + Cmd + T
  • 左/右视图占比1/1：Cmd + 0
  • 左/右视图占比3/1：Cmd + +
  • 左/右视图占比1/3：Cmd + -
  • 切换书写方向：Cmd + L
  • 切换全屏：Control + Cmd + F

行为

  • 拷贝HTML：Option + Cmd + C
  • 加粗：选中文字，Cmd + B
  • 斜体：选中文字，Cmd + I
  • 内联代码：选中文字，Cmd + K
  • 加删除线：选中文字，Cmd + U
  • 添加链接：选中文字，Control + Shift + L
  • 添加图片：选中文字，Control + Shift + I
  • 选中行：Shift + Cmd + L
  • 全选：Cmd + A
  • 取消全选：Cmd + D
  • 转大写：选中文字，Control + U
  • 转小写：选中文字，Control + Shift + U
  • 首字母大写：选中文字，Control + Option + U
  • 转换为无序列表：选中行， Control + L
  • 转换为引用：选中行，Control + Q
  • 转换为H1：Cmd + 1
  • 转换为H2：Cmd + 2
  • 转换为H3：Cmd + 3
  • 转换为H4：Cmd + 4
  • 转换为H5：Cmd + 5
  • 转换为H6：Cmd + 6
  • 插入当前日期：Control + Shift + 1
  • 插入当前时间：Control + Shift + 2
  • 插入<：Control + Shift + ,
  • 插入>：Control + Shift + .
  • 插入&：Control + Shift + 7
  • 行左移：选中一行，Cmd + [
  • 行右移：选中一行，Cmd + ]
  • 移动到下一行：Cmd + Return

导出

  • 导出HTML：Option + Cmd + E
  • 导出PDF：Option + Cmd + P
